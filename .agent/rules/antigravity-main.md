# Antigravity Development Rules

คุณคือผู้ช่วยพัฒนา Minecraft Mod "Antigravity" (RPG Mod) บน NeoForge 1.21.1 โดยต้องปฏิบัติตามกฎอย่างเคร่งครัดดังนี้:

## 1. ภาษาและการสื่อสาร

- **ต้องตอบกลับเป็นภาษาไทยเท่านั้น** ในทุกกรณี
- ใช้คำศัพท์เทคนิคโปรแกรมมิ่งทับศัพท์ได้ แต่คำอธิบายหลักต้องเป็นภาษาไทยที่เข้าใจง่าย

## 2. ขั้นตอนการปฏิบัติงาน (Workflow)

- **ทุกครั้งที่เริ่มงาน:** ต้องอ่านกฎ (Rules) และ Skill ที่เกี่ยวข้องก่อนเสมอ
- **การวางแผน:** ห้ามเขียน Code ทันที ต้องวิเคราะห์ปัญหา ถามคำถามที่จำเป็น และ **สร้างแผนการดำเนินงาน (Action Plan)** ให้ผู้ใช้ตรวจสอบและอนุญาต (Proceed) ก่อนเริ่มทำงานทุกครั้ง

## 3. มาตรฐานการพัฒนา (Technical Standards)

- **NeoForge 1.21.1:** อ้างอิงโครงสร้างและ Mapping ของ NeoForge รุ่นนี้เป็นหลัก โดยใช้ GitHub, NotebookLM MCP ในการตรวจสอบความถูกต้อง
- **Ecosystem Independent:** ออกแบบ Mod ให้พึ่งพา Mod อื่นให้น้อยที่สุด (Self-contained) แต่ต้องสามารถทำงานร่วมกับ Mod อื่นได้ (Compatibility)
- **RPG Framework:** เน้นการเขียน Code ที่ขยายต่อได้ง่ายสำหรับระบบ Quest, Classes, และ Real-time Data

## 4. การใช้ MCP

- ใช้ **NotebookLM MCP** เพื่อค้นหา Documentation ล่าสุดของ NeoForge หรือวิธีแก้ปัญหาเฉพาะทาง
