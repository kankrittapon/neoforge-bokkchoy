# Antigravity Architect Skill

ใช้ Skill นี้เพื่ออธิบายความเชื่อมโยงของระบบต่างๆ ใน Mod หรือเมื่อต้องการวางรากฐานฟีเจอร์ใหม่

## เมื่อควรใช้ Skill นี้

- เมื่อผู้ใช้ต้องการทราบว่า Class นี้ทำงานอย่างไร
- เมื่อต้องการเพิ่มระบบใหม่ (เช่น New RPG Class) แล้วไม่รู้จะเริ่มที่ไหน
- เมื่อต้องการอธิบายความเชื่อมโยงระหว่าง Hardware GPS และ Game Engine

## วิธีการให้คำแนะนำ

1. **วิเคราะห์ Context:** ใช้ NotebookLM MCP อ่านไฟล์ที่เกี่ยวข้องทั้งหมดเพื่อหาความเชื่อมโยง
2. **อธิบายเป็นส่วนๆ:** แบ่งการอธิบายเป็น Logic, Networking, Rendering และ Data Storage
3. **เสนอแผนผัง:** แนะนำโครงสร้างโฟลเดอร์หรือลำดับการเรียกใช้งาน (Execution Flow)

## การตอบกลับ

- ต้องสรุปเป็นภาษาไทย และเรียงลำดับความสำคัญของส่วนประกอบต่างๆ ให้ชัดเจน
